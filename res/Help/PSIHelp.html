All interferograms must have the mirror in the same spot and be same size image with the mirror also being the same size in all the images.
  1. If the phase difference is constant and you know it set it into the "phase angle between images field". This will then set the phase for each image after it is loaded. If you do not know it then ignore that field and select the "Phase is approximate or unknown radio button".
  2. Select "Browse" to add list of interferograms"
  3. If phase is unknown press the "Compute Phases" button. That button will show only if you have selected "Phase is approxiamte or unknown".
  4. Phases will be computed and refined by an iterative process that hopefully will converd.
  5. Review the phase plot graph to see if it is reasonable. The phase angle and the difference from the previous interferogram is plotted on a spiral."
  6. Press the "Phase is known" radio button and then press OK to compute the surface."
  7. A diagrom of the surface will be shown in black and white with a yellow circle that should outline the mirror." You can adjust the outline to better fit the mirror."
  8. Press ok and a surface will be created and added to the surface display.
  • For additional help see the video here.
  • Large interferograms can be resized to smaller images to help speed up the process. Use the resize check box to enable resizing.
  • If compute phase fails to converge sometime it will if you use more iterations or a slightly large tolerance.
